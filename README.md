# Experiments for the RC-BE(α) algorithm
Cristiano Migali, Marco Mussi, Gianmarco Genalti, and Alberto Maria Metelli

## How to Run the Experiments
The code to run the experiments is distributed as a Jupyter Notebook: `rcbealpha_experiments.ipynb`.
The required packages are listed in the _Import_ section of the notebook.
